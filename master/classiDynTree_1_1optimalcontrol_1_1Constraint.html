Detailed Description

The Constraint virtual class definition.

Warning
This class is still in active development, and so API interface can change between iDynTree versions.Inherit publicly from this class to define a constraint of an optimal control problem.

MatrixDynSize &  jacobian 
)
virtual

Evaluate the constraint jacobian with respect to the state variables This methods evaluates the partial derivative of the constraint wrt to the state.

By default, the implementation return false. The user needs to override this method if the constraint will be used, for example, within an optimal control problem that will be solved using an optimizer that needs this information (i.e. Ipopt).

Parameters
[in]timeThe time at which the constraint jacobian is evaluated.
[in]stateThe state at which the constraint jacobian is evaluated.
[in]controlThe control at which the constraint jacobian is evaluated
[out]jacobianThe jacobian right hand side value.
Returns
True if successful. False otherwise (or by default).

Reimplemented in iDynTree::optimalcontrol::LinearConstraint.
