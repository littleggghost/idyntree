Get the base proper (i.e.

including gravitational acceleration) and classical (i.e. not spatial) acceleration

\[ {}^b a_b \]

Note
the classical -> spatial acceleration conversion is dependent on the base velocity, so it is important to set the base velocity before calling this method

If you have doubt of this formula, check Featherstone - Rigid Body Dynamics Algorithms - 2008 - Section 2.11
