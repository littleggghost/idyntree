Detailed Description

Class providing the raw coordinates and semantics for any angular motion vector.

An angular motion vector can be used to describe any angular velocity or acceleration, and implement the adjoint transformations common to these geometric relations.

The angular motion vector is also used to represent a generic element of the so(3) spaces.
