)

For a given regressor and a given dataset, calculate the essential parameters.

The essential parameters are defined in: PHAM, C. M.; GAUTIER, M. Essential parameters of robots. In: Decision and Control, 1991., Proceedings of the 30th IEEE Conference on. IEEE, 1991. p. 2769-2774.

Returns
0 if all went well, -1 if the size of the regressor and the dataset does not match
